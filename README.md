## How This Site Is Built

A single static `index.html` file with embedded CSS and JavaScript. No build step. No frameworks. No external dependencies beyond Google Fonts (Fraunces and Manrope). Designed to load fast on any device and to be readable on slow connections — because recruiters often open candidate portfolios on the move.

### Deploying to GitHub Pages (free hosting)

If you want to host this site on GitHub Pages, here is the step-by-step:

1. Create a GitHub account at github.com if you do not have one.
2. Create a new public repository named exactly `<your-username>.github.io` (for example, `destrylias.github.io`). This special naming convention enables free hosting at `<your-username>.github.io`.
3. Upload `index.html` and this `README.md` to the repository root. You can use the GitHub web interface — no Git command line required.
4. Go to repository **Settings → Pages**. Under "Source," select **Deploy from a branch**, choose `main` branch, root folder. Save.
5. Wait 1–3 minutes. Your site will be live at `https://<your-username>.github.io`.

### Alternative: Netlify (also free, drag-and-drop)

1. Create an account at netlify.com.
2. Drag the folder containing `index.html` onto the Netlify dashboard.
3. Netlify gives you a URL like `https://destrylia-portfolio.netlify.app` immediately.
4. You can later connect a custom domain if desired.

### Editing the Site

Open `index.html` in any text editor (Notepad, VS Code, Sublime Text). All content is plain HTML. To update a project description or your contact details, search for the relevant text and edit it directly. Save. Re-upload to GitHub or Netlify.
